## Workflow Example

### Police Call - Burglary in Progress

1. **Select Service:** Police
2. **Location:** "123 High Street, Edinburgh"
3. **Phone:** "07700 900123"
4. **Name:** "John Smith"

**Police-Specific:**
- **What's happening:** "Two males breaking into shop through rear door"
- **Weapons:** "Unknown"
- **Injuries:** "No injuries"
- **Suspect Description:** "Two males, dark clothing, wearing hoodies, approx 20s"
- **Caller Safe:** "Yes, I am safe - watching from across the street"

**Additional Notes:** "Caller is business owner, has CCTV"

Click **Save Call Record**

## Tips for Efficiency

1. **Create CAD First** - For deployable incidents, create the CAD immediately
2. **Copy CAD Reference** - Add it to the call record for tracking
3. **Use Dropdowns** - Faster than typing, ensures consistency
4. **Fill While Talking** - Type as caller speaks (when safe to do so)
5. **Save Frequently** - Don't lose information if call disconnects

---

## Integration with CAD System

The call taking form works seamlessly with the main CAD system:

1. Take emergency call using form
2. Create CAD from information gathered
3. Link CAD reference back to call record
4. Both records now connected for audit trail

This creates complete documentation from initial call to incident resolution.
